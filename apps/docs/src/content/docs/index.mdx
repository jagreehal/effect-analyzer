## What it changes for you

<CardGrid>
  <Card title="PR reviews catch real changes" icon="approve-check">
    Text diffs say *"3 lines changed"*. The semantic diff says *"retry removed, `DbService` added,
    error path now uncaught"*. Stop merging behaviour drift on autopilot.
  </Card>
  <Card title="Onboard without reading every yield" icon="seti:plan">
    New teammates open the interactive HTML viewer for a function and see services, layers, error
    paths and concurrency at a glance — instead of tracing generators line-by-line.
  </Card>
  <Card title="Docs that can't lie" icon="puzzle">
    Service maps and railway diagrams generated straight from the AST. Refactor freely; the
    diagrams move with the code, not with a stale Notion page.
  </Card>
  <Card title="Catch hidden complexity in CI" icon="warning">
    Cyclomatic, cognitive, path count, nesting depth, parallel breadth — fail the build when a
    function quietly crosses your team's thresholds.
  </Card>
</CardGrid>

## Quick Start

```bash
# Install
npm install effect-analyzer

# Analyze a file
npx effect-analyze ./src/my-program.ts

# Generate a railway diagram
npx effect-analyze ./src/my-program.ts --format mermaid-railway

# Compare versions
npx effect-analyze HEAD:program.ts program.ts --diff
```
